# Guided Learning 🧭

Guided Learning is a multi-platform AI-guided learning system that transforms your workspace into an adaptive learning environment. It tracks your progress, identifies knowledge gaps, and evolves a persistent knowledge vault using frameworks like **Gap Analysis** and **Bloom's Taxonomy**.

It is compatible with major AI coding assistants: **Gemini CLI**, **Claude Code**, **OpenCode**, and **Codex / GitHub Copilot**.

## ✨ Features
- **Adaptive Questioning**: Automatically detects what you don't know and asks the "One Best Next Question."
- **Persistent Vault**: Keeps your learning state in human-readable Markdown (`goal.md`, `roadmap.md`, `memory.md`).
- **Framework-Driven**: Dynamically switches between learning frameworks (e.g., Gap Analysis, Bloom's Taxonomy, Kolb's Learning Cycle) based on your current expertise level.
- **Session Sync**: Distills entire learning conversations into atomic wiki notes and roadmap updates.

## 🛠️ Installation & Setup

Guided Learning can be set up in a few simple steps depending on your preferred AI tool.

See the **[INSTALL.md](INSTALL.md)** file for detailed instructions on:
1. **Gemini CLI**: Installing as a native skill.
2. **Claude Code**: Configuring with `CLAUDE.md`.
3. **OpenCode**: Configuring with `agent.md`.
4. **Codex / GitHub Copilot**: Configuring with `.github/copilot-instructions.md`.

## 🚀 Getting Started

Once set up, you can start learning in any workspace folder:
1. **Initialize**: Define your primary goal, constraints, and initial milestones.
2. **Interact**: Let the AI guide your learning by identifying knowledge gaps and asking targeted questions.
3. **Process**: Distill the conversation, update memory logs, and evolve your roadmap.
4. **Review**: Track completed milestones and visualize your learning progress.
